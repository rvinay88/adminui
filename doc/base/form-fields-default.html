A label, an input text and any feedback messages are grouped into a div.form-group `.form-group` can have additional classes as below

  1. .has-feedback - to reveal the .help-text block hidden under the input control
  2. .has-error
  3. .has-info
  4. .has-warning
  5. .has-success

`.help-text` It can be preloaded with error messages, which are then revealed when we add `.has-feedback` class to the `.form-group`
